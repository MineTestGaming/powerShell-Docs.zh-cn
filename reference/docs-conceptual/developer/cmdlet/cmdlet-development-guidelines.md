# <a name="cmdlet-development-guidelines"></a>Cmdlet 开发指南

本节中的主题提供了可用于生成格式正确的 cmdlet 的开发指南。 通过利用 Windows PowerShell 运行时提供的通用功能并遵循这些准则，你可以最大程度地开发强大的 cmdlet，并为用户提供一致的体验。 此外，您还可以减少测试负担，因为一般功能不需要重新测试。
